All About Type Writer Plugin

What is Type Writer Effect?

Type writer effect is a way of showing text just like old type writer print character by character on a paper. In the same way the text in displayed by using this plug-in.

How to configure?

You can configure all of its attributes by just passing its value at the time of initialization.

Initialization: 
                    

                        $( "div" ).typewriter();
Initialization with attributes: 
                    

                        $( "div" ).typewriter({
                            interval: 80,
                            color: "green",
                            fontSize: "18px"
                        });
  • interval: is the time interval to show each character. Its value is passed in microseconds like 80, 100
  • color: is the color of the text. You can pass color using Color Name like RED or HEX Value like #F0F8FF
  • fontSize: is the font size of the text. You can pass any value like 12px, 14px etc
